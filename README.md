# Personal Portfolio

A high-end editorial minimalism portfolio website built with TypeScript and Tailwind CSS.

## Features

- **Editorial Minimalism**: Gallery-like aesthetic with emphasis on whitespace and typography
- **Four Main Sections**:
  - Identity (Hero): Name, affiliations, and animated marquee
  - Focus: Experience and projects in split-screen layout
  - Toolkit: Borderless skills table
  - Connection: CTA and social links
- **Micro-interactions**:
  - Grain overlay for texture
  - Magnetic links with cursor attraction
  - Scroll progress indicator
  - Marquee that pauses on hover

## Tech Stack

- TypeScript
- Vite
- Tailwind CSS
- Vanilla JavaScript (no frameworks)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
